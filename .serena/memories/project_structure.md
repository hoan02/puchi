# Project Structure

- `src/app/` — Next.js app directory (routes, layouts, API endpoints)
- `src/components/` — UI and feature components (including onboarding/welcome flows)
- `src/constants/` — Static values (navigation, paths, themes)
- `src/hooks/` — Custom React hooks
- `src/i18n/` — Internationalization logic
- `src/lib/` — Helpers and utility libraries
- `src/services/` — Service logic (dictation, lessons, user)
- `src/store/` — State management (Zustand)
- `src/styles/` — CSS and font definitions
- `src/types/` — TypeScript type definitions
- `src/utils/` — Utility functions

Entry point scripts and configuration files are in the project root (e.g., `package.json`, `tsconfig.json`, `eslint.config.mjs`).
