# Project Purpose and Tech Stack

**Puchi** is an innovative platform designed to help users master Vietnamese, focusing on mobile and on-the-go learning. It features:
- Next.js 15 (App Router, Server Actions)
- React 19
- TypeScript
- Tailwind CSS (with dark/light mode)
- Shadcn UI component system
- Clerk for authentication
- Internationalization (i18n)
- Analytics integration
- Optimized for performance and responsive design

The project is developed on Windows, but uses standard Node.js/Next.js tooling.
